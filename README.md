# Spaceshield_Hack_2026_PTR_DLD_Digital_Twin_Mission
Projekt rakietowego systemu wynoszenia małego satelity na niską orbitę, obejmujący projekt rakiety, profil lotu oraz sekwencję separacji, tworząc cyfrowego bliźniaka, który symuluje i odwzorowuje wszystkie te elementy oraz umożliwia analizę przebiegu misji, z uwzględnieniem optymalizacji masy rakiety przy jednoczesnym osiągnięciu orbity.
